주차 이용 안내

1. 입주자 차량 등록

등록 방법

  1. 주차 메뉴 → 입주자 차량 조회 → 우측 상단 차량 추가 버튼 클릭
  2. 차량 번호, 차량 모델, 차량 종류(승용차/SUV/화물차 등) 입력
  3. 자동차 등록증 사진 또는 PDF 파일 첨부 후 제출
  4. 관리자 승인 완료 시 입출차 및 티켓 서비스가 자동 활성화됩니다.

필요 서류

  • 자동차 등록증 (사진 또는 PDF) — 차량 번호·소유자명이 반드시 확인 가능해야 합니다.

승인 절차

  1. 서류 제출 후 승인 대기 상태로 등록됩니다.
  2. 관리자 검토 후 승인 또는 반려 처리됩니다.
  3. 반려 시 반려 사유를 확인하고 서류를 보완하여 재신청할 수 있습니다.

주의 사항

  • 차량은 무료로 1대만 등록 가능합니다. 2대부터는 유료로 관리사무소로 문의해 주세요.
  • 이미 다른 세대에 등록된 차량 번호는 중복 등록할 수 없습니다.
  • 차량 번호 변경 시 기존 차량을 삭제하고 새로 등록해야 합니다.
  • 승인 완료된 차량은 입주자가 직접 삭제할 수 없습니다. 삭제가 필요한 경우 관리사무소(000-0000)에 문의해 주세요.

2. 방문 차량 사전 예약

예약 방법

  1. 주차 메뉴 → 사전 차량 예약 → 예약하기 버튼 클릭
  2. 방문 차량 번호, 방문 목적, 방문 예정 시각 입력 후 등록
  3. 방문 차량 입차 시 사전 예약 정보가 자동으로 연결됩니다.

취소 방법

  1. 주차 메뉴 → 사전 차량 예약에서 예약 목록 확인
  2. 취소할 예약의 취소 버튼 클릭
  3. 이미 입차 완료된 예약은 취소할 수 없습니다.

주의 사항

  • 방문 예정 시각 기준 전후 30분 이내에 입차해야 예약이 자동 연결됩니다.
  • 30분을 초과하여 입차한 경우 예약이 연결되지 않으므로 예약을 다시 등록해야 합니다.
  • 예약 없이 입차한 방문 차량은 수동 입차로 처리되며, 방문 차량 관리에서 직접 세대를 연결해야 합니다.
  • 동일 차량 번호로 중복 예약은 불가합니다.

3. 방문 차량 관리

미등록 방문 차량 세대 연결

  1. 사전 예약 없이 입차한 방문 차량을 내 세대 방문자로 직접 등록할 수 있습니다.
  2. 주차 메뉴 → 방문 차량 관리 클릭
  3. 차량 번호 끝 4자리 입력 후 조회
  4. 해당 차량 선택 후 내 방문자로 등록 버튼 클릭

주의 사항

  • 사전 예약 없이 입차한 방문 차량만 조회됩니다.
  • 이미 다른 세대에 등록된 차량은 등록할 수 없습니다.
  • 출차 완료된 차량은 등록할 수 없습니다.

4. 주차 티켓 사용 방법

티켓 종류 및 발급 수량

  • 1시간권 — 1시간 무료 주차 제공 (60분 차감) / 매월 60장 발급
  • 1일권 — 24시간 무료 주차 제공 (1,440분 차감) / 매월 6장 발급

적용 방법

  1. 주차 메뉴 → 티켓 등록에서 차량 번호 끝 4자리 입력 후 조회
  2. 현재 주차 중인 차량 목록에서 대상 차량 선택
  3. 적용할 티켓 종류 선택 후 수량 조절
  4. 적용 버튼 클릭 — 출차 전까지 수정 및 취소 가능

유의 사항

  • 티켓은 매월 1일 세대별 자동 발급됩니다.
  • 미사용 티켓은 다음 달로 이월되지 않으니 월말 전에 사용하세요.
  • 주차 시간이 티켓 적용 시간을 초과한 경우 초과분은 유료로 정산됩니다.

5. 주차 기록 조회

조회 방법

  1. 주차 메뉴 → 월별 주차 기록 클릭
  2. 상단 년/월 선택으로 원하는 달의 주차 기록 조회
  3. 입주자 차량 및 방문 차량의 입출차 시간, 정산 내역을 확인할 수 있습니다.

정산 내역 안내

  • 무료 — 티켓 미사용, 무료 주차 시간 이내 출차
  • 티켓 사용 — 티켓이 적용된 경우
  • 초과 N시간 N분 — 티켓 적용 후에도 초과 주차가 발생한 경우
  • 주차 중 — 아직 출차하지 않은 차량

6. 전기차 충전소 위치

전기차 충전소 위치 : 101동/ 103동/ 106동 주차장에 있습니다.
