# 🏠 Hometalkonepass
> 공동주거시설 입주민 전용 통합 관리 플랫폼

<br>

## 📌 프로젝트 소개

오피스텔·아파트 거주 경험에서 출발한 프로젝트입니다.
종이 공지 누락, 관리비 고지서 분실, 주차 티켓 수동 계산 등
실생활에서 직접 겪은 불편함을 해소하기 위해 기획했습니다.

분산되어 있던 공동주거시설의 관리 기능을 하나의 플랫폼으로 통합하여
**입주민 편의성과 운영 효율을 높이는 것**을 목표로 합니다.

<br>

## ✨ 담당 기능 — 주차 모듈

### 입주자
- 차량 등록 (증빙 서류 첨부 필수)
- 반려 시 사유 확인 및 재등록
- 방문 차량 사전 예약 (미래 날짜 예약 가능)
- 주차 중인 방문 차량에 티켓 등록 (1시간권 / 1일권)
- 세대별 주차 이력 월별 조회

### 관리자
- 차량 등록 서류 승인 / 반려 (반려 사유 필수 입력)
- 세대별 티켓 자동 발급 / 수동 발급 / 선택 세대 발급
- 전체 주차 기록 조회

### 경비원
- 차량 번호 뒷자리 4자리 퀵 서치
- 실시간 입/출차 처리 및 신원 식별
- 수동 입차 처리 (미등록 차량 — 택배, 방문 등)

### 홈 화면
- 공지 · 일정 · 커뮤니티 대시보드 연동

<br>

## 🔄 주차 시스템 흐름

```
[1단계] 입주민 차량 등록 / 방문 차량 사전 예약
        ↓
[2단계] 경비원 퀵 서치 → 입주자 / 예약 / 미등록 자동 분류 → 입차 승인
        ↓ 입차 알림 발송
[3단계] 입주민 티켓 적용 (1시간권 / 1일권)
        ↓
[4단계] 경비원 출차 처리 → 자동 정산
        ↓ 출차 알림 발송
```

**요금 정책**
- 기본 요금: 30분당 1,000원 (30분 단위 올림)
- 입차 후 10분 이내 출차 시 무료
- 티켓 우선 적용 후 초과 시간만 과금
